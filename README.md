# OnlyofficeS3Wrapper

ONLYOFFICE Helper Gem for S3. Used in QA

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'onlyoffice_s3_wrapper'
```

And then execute:

```shell script
bundle
```

Or install it yourself as:

```shell script
gem install onlyoffice_s3_wrapper
```
